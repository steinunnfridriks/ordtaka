# Orðtökutól/Lexicon Acquisition Tool

Orðtökutólið er notað til þess að safna saman orðum úr [Risamálheildinni](https://malheildir.arnastofnun.is/) sem vantar í önnur textasöfn, fyrst og fremst [Beygingarlýsingu íslensks nútímamáls (BÍN)](bin.arnastofnun.is/) og hins vegar [Íslenska nútímamálsorðabók (ISLEX)](https://islenskordabok.arnastofnun.is/).
